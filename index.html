About

This is a physics particle simulator made by Qiyuan Chen. This was designed as a way to simulate Brownian motion, where particles move about randomly in a fluid due to collisions with each other.


Using the settings, you can change a wide range of variables, ranging from the number, type and speed of the particles. However please make sure you do not include too many particles or very large particles, because that could potentially cause the program to lag out.


Apart from that, enjoy messing around with this simple simulation. If you want to see my other projects or know more about me, make sure to check out http://qchen.netlify.app/
